# GitTest

### 基本操作

+ git init：初始化仓库
+ git status：查看仓库的状态
+ git add：向暂存区添加文件
+ git commit：保存仓库的历史纪录
  + 记述简单提交信息
    + -m “提交信息”
  + 记述详细提交信息
    + 第一行：用一行文字简述提交的根本更改内容
    + 第二行：空行
    + 第三行以后：记述更改的原因和详细内容
+ git log：查看提交日志
  + --pretty=short：只显示提交信息的第一行
  + -p：显示文件的改动
+ git diff：查看更改前后的差别
  + git diff HEAD：查看工作树和最新提交的差别
